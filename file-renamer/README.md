# File Renamer

バッチファイルリネームツール（Python/tkinter）

## 起動方法

```bash
python3 file_renamer.py
```

## 機能

- **基本**: プレフィックス/サフィックス追加、テキスト検索・置換、拡張子変更
- **正規表現**: `re.sub` ベースの高度な置換（グループ参照 `\1` も可）
- **大文字/小文字**: 全大文字/小文字/タイトルケース変換
- **連番**: 開始番号・ステップ・桁数・位置（前/後）を指定してナンバリング
- ファイルリスト: リネーム前後をリアルタイムプレビュー
- ドラッグ&ドロップ対応（tkinterdnd2 インストール時）

## 依存ライブラリ

- Python 3.x 標準ライブラリのみ（tkinter）
- ドラッグ&ドロップを使う場合: `pip install tkinterdnd2`
